\paragraph{1. Equivalence of negative log probability and logistic loss (10 points)} After replacing the label set from $\left\{ 0,1 \right\}$ to
$\left\{ -1,1 \right\}$, we introduced the log loss
\begin{align*}
    D_{\log}(y, \vx; M) = \frac{1}{\log 2} \log(1+\exp(-s(y, \vx; M))),
\end{align*}
as an alternative to the logistic regression distance function above. Show that
these two are equivalent up to a constant multiplication for logistic
regression.

\vspace{5mm}

\paragraph{2. Hinge loss gradients (10 points)} Unlike the log loss, the hinge loss, defined below, is not
differentiable everywhere:
\begin{align*}
    D_{\hinge}(y, \vx; M) = \max(0, 1 - s(y, \vx; M)).
\end{align*}
Does it mean that we cannot use a gradient-based optimization algorithm for
finding a solution that minimizes the hinge loss? If not, what can we do about
it?

\vspace{5mm}

\paragraph{3. Model Selection (20 points due to importance)} Consider that we are learning a logistic regression $M^{(1)}$ and a perceptron $M^{(2)}$, and we have three dataset partitions: a training set $D_\text{train}$, a validation set $D_\text{val}$, and a test set $D_\text{test}$.

The two models are iteratively optimized on $D_\text{train}$ over $T$ steps, and now we have $T$ logistic regression parameter configurations (i.e. weights and biases) $M^{(1)}_1, M^{(1)}_2, \dots, M^{(1)}_T$ and $T$ perceptron configurations $M^{(2)}_1, M^{(2)}_2, \dots, M^{(2)}_T$, all with different parameters.

We now evaluate the expected cost for all the $2T$ models on training set, validation set, and test set.  So we have $6T$ quantities $\tilde{R}^{(i)}_{\text{train}, t}$, $\tilde{R}^{(i)}_{\text{val}, t}$, $\tilde{R}^{(i)}_{\text{test}, t}$ where $i=1,2$ and $t=1,\dots,T$.

\begin{enumerate}[label=(\alph*)]
	\item Which $i$ and $t$ should we pick as the best model? (10 points)
	\item How should we report the generalization error? (10 points)
\end{enumerate}
